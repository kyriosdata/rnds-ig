Extension: Informações Complementares de Indivíduos Não Identificados

Informações complementares necessárias ao Contato Assistencial na hipótese do indivíduo não poder ser identificado.

The official URL for this extension is: 

http://www.saude.gov.br/fhir/r4/StructureDefinition/BRIndividuoNaoIdentificado-1.0

Status: active
Extension maintained by: Rede Nacional de Dados em Saúde

Context of Use

This extension may be used on the following element(s):

  • Composition.subject
  • Encounter.subject
  • Condition.subject
  • Procedure.subject
  • Immunization.patient

Usage info

Usage:

  • Use this Extension: Imunobiológico Administrado, Procedimento Realizado, Conjunto Mínimo de Dados, Indivíduo, CIAP2 Avaliado, Contato Assistencial, Registro de Imunobiológico Administrado, CID10 Avaliado, Conjunto Mínimo de Dados
